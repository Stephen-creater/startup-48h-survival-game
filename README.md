# 创业者48小时生存实验

创业模拟互动游戏 - 你能活多久？

## 项目简介

这是一款赛博朋克风格的互动叙事游戏，模拟创业者在辞职后48小时内的生存挑战。通过资源管理、道德困境和分支选择，让用户体验创业的残酷真实。

## 核心特性

- **4维资源管理**：金钱、时间、精力、人脉
- **6个关键节点**：0h, 8h, 16h, 24h, 36h, 48h
- **6种结局**：破产、崩溃、放弃、生存、成功、不择手段
- **残酷真实叙事**：无美化、有冲击力的创业体验
- **赛博朋克UI**：霓虹绿、故障艺术、打字机效果
- **音效系统**：打字音、警告音、选择音、资源消耗音
- **移动端优化**：完整的响应式设计，支持手机和平板
- **分享功能**：生成精美的结局分享图片

## 技术栈

- 纯前端实现（HTML + CSS + JavaScript）
- Web Audio API 生成音效
- html2canvas 生成分享图片
- LocalStorage 保存游戏进度和统计数据
- 响应式设计，支持移动端

## 快速开始

1. 直接在浏览器中打开 `index.html`
2. 或使用本地服务器：
   ```bash
   python3 -m http.server 8000
   # 访问 http://localhost:8000
   ```

## 游戏流程

1. **开场动画**：打字机效果展示背景故事
2. **游戏主界面**：资源条 + 剧情 + 选择
3. **选择后果**：显示决策结果和资源变化
4. **结局判定**：根据资源和标记判定结局
5. **分享功能**：生成精美的结局分享图片

## 结局判定逻辑

结局按优先级判定：

1. **放弃**（击败0%）：选择"回去找工作"
2. **崩溃**（击败15%）：精力≤0 或选择"我撑不住了"
3. **破产**（击败35%）：金钱≤0 或选择"我没钱了"
4. **不择手段**（击败99%）：违反竞业协议 + 获得客户
5. **成功**（击败99.2%）：金钱>5000 + 有客户，或金钱>3000 + 有原型 + 交房租
6. **生存**（击败95%）：交了房租或协商房租

## 音效系统

使用 Web Audio API 实时生成音效，无需外部音频文件：

- **打字音**：开场动画时播放
- **选择音**：点击选项时播放
- **资源消耗音**：消耗资源时播放
- **警告音**：资源低于30%时播放

## 移动端优化

- **768px以下**：平板适配，2列资源显示
- **480px以下**：手机适配，单列资源显示
- 触摸优化的按钮尺寸
- 自适应字体大小
- 优化的间距和布局

## 数据统计

游戏会自动保存到 LocalStorage：
- `gameResults`：所有游戏记录
- `latestResult`：最新一次游戏结果

统计数据包括：
- 结局类型
- 存活时间
- 最终资源
- 选择路径
- 时间戳
