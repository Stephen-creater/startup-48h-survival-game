# 创业者48小时生存实验

一个黑镜风格的创业生存模拟游戏。

## 项目简介

这是一个互动式网页游戏，模拟创业者在辞职后48小时内的生存挑战。玩家需要管理有限的资源（金钱、时间、精力、人脉），在一系列艰难的选择中求生存。

## 核心特色

- **黑镜式叙事**：残酷、真实、让人不舒服但印象深刻
- **资源管理**：4维资源系统（💰金钱、⏰时间、⚡精力、👥人脉）
- **多重结局**：6种不同结局，根据玩家选择动态判定
- **道德困境**：没有标准答案的选择，测试玩家的底线
- **社交传播**：内置分享功能，激发用户挑战欲

## 技术栈

- 纯前端实现（HTML + CSS + JavaScript）
- 无需后端服务器
- 使用 LocalStorage 保存游戏进度和统计数据
- 响应式设计，支持移动端

## 游戏流程

1. **开场动画**：黑镜式打字机效果
2. **6个关键节点**：0h → 8h → 16h → 24h → 36h → 48h
3. **资源管理**：每个选择都会影响资源
4. **结局判定**：根据资源和选择标记判定6种结局

## 6种结局

1. **破产**（60%）- 金钱耗尽
2. **崩溃**（20%）- 精力耗尽
3. **放弃**（15%）- 主动放弃
4. **勉强生存**（4%）- 交了房租，继续战斗
5. **初步成功**（0.8%）- 有客户或原型，且有钱
6. **不择手段**（0.2%）- 违反规则但活下来

## 自定义

### 修改剧情

编辑 `js/story.js`，修改 `storyNodes` 数组中的内容。

### 修改结局

编辑 `js/endings.js`，修改 `endings` 对象和 `determineEnding` 函数。

### 修改样式

编辑 `css/style.css` 和 `css/animations.css`。

## 数据统计

游戏会自动保存玩家数据到 LocalStorage：

- `gameResults`：所有游戏记录
- `latestResult`：最新一次游戏结果

可以通过浏览器控制台查看：

```javascript
// 查看所有结果
JSON.parse(localStorage.getItem('gameResults'))

// 查看最新结果
JSON.parse(localStorage.getItem('latestResult'))
```

## 配置小助手微信

在 `js/game.js` 的 `contactAssistant()` 函数中，修改微信号：

```javascript
function contactAssistant() {
  alert('添加小助手微信：[你的微信号]\n\n我们会给你更详细的创业建议和反馈。');
}
```
