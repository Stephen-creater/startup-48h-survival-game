# 创业者48小时生存实验

黑镜式互动创业游戏 - 你能活多久？

## 项目简介

这是一款黑镜风格的互动叙事游戏，模拟创业者在辞职后48小时内的生存挑战。通过资源管理、道德困境和分支选择，让用户体验创业的残酷真实。

## 核心特性

- **4维资源管理**：金钱、时间、精力、人脉
- **6个关键节点**：0h, 8h, 16h, 24h, 36h, 48h
- **6种结局**：破产、崩溃、放弃、生存、成功、不择手段
- **黑镜式叙事**：残酷、真实、有冲击力
- **赛博朋克UI**：霓虹绿、故障艺术、打字机效果

## 技术栈

- 纯前端实现（HTML + CSS + JavaScript）
- 无需后端服务器
- LocalStorage保存游戏进度和统计数据
- 响应式设计，支持移动端

## 快速开始

1. 直接在浏览器中打开 `index.html`
2. 或使用本地服务器：
   ```bash
   python3 -m http.server 8000
   # 访问 http://localhost:8000
   ```

## 游戏流程

1. **开场动画**：黑镜式打字机效果
2. **游戏主界面**：资源条 + 剧情 + 选择
3. **选择后果**：显示决策结果
4. **结局判定**：根据资源和标记判定结局
5. **分享功能**：生成分享文案

## 结局判定逻辑

- **破产**：金钱≤0 或选择"我没钱了"
- **崩溃**：精力≤0 或选择"我撑不住了"
- **放弃**：选择"回去找工作"
- **不择手段**：违反竞业协议 + 获得客户
- **成功**：金钱>5000 + 有客户，或金钱>3000 + 有原型 + 交房租
- **生存**：交了房租或协商房租

## 数据统计

游戏会自动保存到 LocalStorage：
- `gameResults`：所有游戏记录
- `latestResult`：最新一次游戏结果

统计数据包括：
- 结局类型
- 存活时间
- 最终资源
- 选择路径
- 时间戳
