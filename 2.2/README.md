# 投资人模拟器 - 项目说明

## 项目概述

这是一个黑镜式创业社交评分游戏,用于识别和筛选具有"结构外生长力"的年轻创业者(14-22岁)。

## 核心玩法

- 玩家扮演创业者,与AI投资人进行10轮对话
- 每个选择都会被隐藏评分,玩家看不到分数
- 只能通过投资人的态度变化(表情、回复速度、用词)感知表现
- 目标:获得足够高的分数,通过投资人的考验

## 技术实现

- 纯前端实现:HTML + CSS + JavaScript
- 无需后端服务器
- 移动端优先设计
- 黑镜风格视觉设计

## 评分机制

- 初始分数: 3.5分
- 分数范围: 0-5分
- 态度变化:
  - 4.5-5.0分: 热情(😊)
  - 3.5-4.4分: 中立(😐)
  - 2.5-3.4分: 冷淡(😒)
  - <2.5分: 拉黑(😠,游戏结束)

## 结局等级

- **S级** (4.5+分): 获得投资,可留下联系方式
- **A级** (3.5-4.4分): 投资人有兴趣,需要继续观察
- **B级** (2.5-3.4分): 被拒绝,但有潜力
- **C级** (<2.5分): 被拉黑,犯了致命错误

## 关键场景

1. **自我介绍** - 测试是否问题导向
2. **创业动机** - 测试动机的具体性
3. **竞品分析** - 测试深度思考能力
4. **团队问题** - 测试行动力
5. **压力测试** - 测试坚持和应变能力(关键场景)
6. **资源需求** - 测试规划能力
7. **道德困境** - 测试价值观(送命题!)
8. **时间规划** - 测试精益思维
9. **失败经历** - 测试反思能力
10. **最后机会** - 测试自信和独立性

## 裂变机制

1. **评分揭秘** - 游戏结束后显示完整评分轨迹
2. **对比挑战** - "我得了4.2分,你能超过我吗?"
3. **争议话题** - 公开"送命题"引发讨论
4. **社交分享** - 一键复制分享文案
