# 投资人模拟器 - 部署指南

## 快速部署到 Vercel

### 步骤1: 安装 Vercel CLI (如果还没安装)
```bash
npm install -g vercel
```

### 步骤2: 部署
```bash
cd /Users/a1-6/Downloads/起点计划/创业小游戏2/2.2
vercel --prod
```

### 步骤3: 获取链接
部署完成后会得到一个链接,例如: `https://your-project.vercel.app`

---

## 快速部署到 Netlify

### 方式1: 拖拽部署
1. 访问 https://app.netlify.com/drop
2. 将整个 `2.2` 文件夹拖拽到页面
3. 等待部署完成,获取链接

### 方式2: CLI部署
```bash
npm install -g netlify-cli
cd /Users/a1-6/Downloads/起点计划/创业小游戏2/2.2
netlify deploy --prod
```

## 监控和优化

### 添加 Google Analytics
在 `index.html` 的 `<head>` 中添加:
```html
<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=YOUR_GA_ID"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'YOUR_GA_ID');
</script>
```

### 关键指标
- 访问量
- 完成率(玩到最后的比例)
- 各场景的选择分布
- 结局分布(S/A/B/C级比例)
- 分享率

## 常见问题

### Q: 游戏在手机上显示不正常?
A: 检查 `viewport` 设置,确保响应式设计正常

### Q: 如何修改评分规则?
A: 编辑 `js/story.js` 中每个选项的 `value` 值

### Q: 如何添加新的场景?
A: 在 `js/story.js` 的 `scenes` 数组中添加新对象

### Q: 如何修改结局内容?
A: 编辑 `js/scoring.js` 中的 `getEndingContent()` 方法
