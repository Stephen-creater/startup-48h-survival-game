# 项目：创业者48小时生存实验

## 当前状态
黑镜风格创业模拟游戏，纯前端实现。

## 核心文件
- `index.html` - 主页面结构
- `css/style.css` - 黑镜赛博朋克风格
- `js/game.js` - 游戏核心逻辑
- `js/audio.js` - 音效系统
- `js/story.js` - 剧情数据
- `assets/sounds/mixkit-hard-laptop-keyboard-typing-2538.wav` - 打字音效

## 当前问题
**布局需求**：用户要求图片+文字作为左侧整体组件，资源条作为右侧组件，两组件并排。

**现状**：资源条在外侧，未按要求紧贴文字右侧。

**解决方案**：
1. 修改HTML：将图片+文字包在一个容器内
2. 资源条独立为右侧组件
3. 两组件用flex并排

## 设计原则
- 极简代码
- 无废话
- 黑镜风格：黑色背景+霓虹绿(#00ff41)+霓虹红(#ff0844)
