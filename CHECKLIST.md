## 🐛 常见问题预案

### 问题1：游戏打不开
**检查**：
- 浏览器控制台是否有错误
- 所有JS文件是否正确加载
- 是否有CORS错误

**解决**：
- 清除浏览器缓存
- 检查文件路径
- 确保所有文件都已上传

### 问题2：资源条不显示
**检查**：
- CSS文件是否加载
- 浏览器兼容性

**解决**：
- 检查CSS文件路径
- 尝试不同浏览器

### 问题3：分享功能不工作
**检查**：
- 浏览器是否支持clipboard API
- 是否有权限问题

**解决**：
- 已实现降级方案
- 提示用户手动复制

### 问题4：移动端显示异常
**检查**：
- viewport设置
- 响应式CSS

**解决**：
- 检查meta标签
- 测试不同屏幕尺寸
