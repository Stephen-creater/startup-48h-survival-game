## 🎮 游戏特色

### 黑镜式设计元素：
1. **残酷的真实感** - 每个选择都有代价
2. **资源焦虑** - 看着资源条一点点耗尽
3. **道德困境** - 违反竞业协议 vs 从零开始
4. **压迫感** - 48小时倒计时
5. **社会实验感** - "只有X%的人能活过24小时"

### 6种结局：
1. **破产**（60%）- 金钱耗尽
2. **崩溃**（20%）- 精力耗尽
3. **放弃**（15%）- 主动放弃
4. **勉强生存**（4%）- 交了房租
5. **初步成功**（0.8%）- 有客户且有钱
6. **不择手段**（0.2%）- 违反规则但活下来
